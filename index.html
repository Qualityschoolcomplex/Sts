App Permissions

Opens the device's own permission screens — browsing what's there is up to the user.

A regular installed app — even a native one, not just a web page — cannot read what permissions other apps hold. That data is only exposed to the OS's own Settings app. These buttons open those built-in screens so the user can look themselves.

Privacy Dashboard / Permission Manager

Browse every app grouped by permission type (camera, location, microphone, etc.) — the device's own built-in view.

A specific app's permissions

Android: enter the app's package name, e.g. com.whatsapp or com.dreamapplications.shopdesk.
Windows: this field isn't used — desktop (Win32) apps don't have a per-app permissions page the way mobile/UWP apps do, so the button below just opens the general Privacy hub instead.
