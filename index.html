Settings Launcher

A tiny Windows app that opens Settings the instant you run it — no installation, no admin rights needed.

Download OpenSettings.exe
First run only: since this isn't a code-signed, widely-downloaded app, Windows SmartScreen or Chrome's download check may show a warning. Click "More info" → "Run anyway" (SmartScreen) or keep the file when Chrome flags it as uncommon. This is standard for any small unsigned tool, not a sign of a problem.
